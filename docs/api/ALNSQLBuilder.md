# ALNSQLBuilder

- Kind: `interface`
- Header: `src/Arlen/Data/ALNSQLBuilder.h`

Fluent SQL builder for `SELECT`/`INSERT`/`UPDATE`/`DELETE` with expression-safe composition and deterministic SQL output.

## Typical Usage

```objc
NSError *error = nil;
ALNSQLBuilder *builder = [[[ALNSQLBuilder selectFrom:@"users" columns:@[@"id", @"email"]]
    whereField:@"status" equals:@"active"]
    orderByField:@"id" descending:NO];
NSString *sql = [builder buildSQL:&error];
NSArray *params = [builder buildParameters:&error];
```
